About Me.

I’m a Software Engineering graduate from Misr International University, passionate about building impactful and efficient software solutions. During my academic journey, I completed internships with Basata and Singleclic, where I gained hands-on experience in real-world development environments, contributing to various backend and frontend tasks.

For my graduation project, I led the development of CodePure, a Visual Studio Code extension designed to detect code smells using software metrics and machine learning. The tool analyzes code structure, identifies quality issues, and provides developers with insights and suggestions for cleaner, more maintainable code.

I’m continuously exploring ways to merge software engineering principles with AI to create tools that solve real-world developer problems.
